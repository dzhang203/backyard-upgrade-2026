# backyard-upgrade-2026

Planning, design, and regulatory notes for a 2026 backyard remodel at 318 24th Ave, San Mateo, CA.

**Scope:** Expand paved area, add a Purple Leaf 10×16 hardtop gazebo near the east fence, resurface under the gazebo with porcelain pavers, install rain-chain + dry-well drainage, and add a native / oak-woodland-style fence-line planting.

## Locked Design Decisions

- **Gazebo:** Purple Leaf 10×16 hardtop, aluminum frame, galvanized steel double roof (eave 82", peak 114")
- **Placement:** 3 ft from east fence (roof line); pavers 1.5 ft from fence
- **Paver field:** 18 ft × 12 ft, 2 ft × 2 ft porcelain pavers, 2 cm thick, rectified-edge, dry-laid on compacted aggregate
- **Surround:** 1" Lynn Creek smooth river rock on all four sides
- **Drainage:** Cup-style rain chains at each of four corners → subsurface dry wells (2×2×2.5–3 ft, geotextile-lined, #57 stone)
- **Corner cut-outs:** 2 ft × 4 ft at each corner post (footing + dry well in adjacent cells)

See individual docs for the reasoning behind each decision.
