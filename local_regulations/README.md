## Key Findings Summary

1. **Permit required:** 160 sq ft gazebo (10×16) exceeds the 120 sq ft permit-free threshold. Submit via [City Online Permit Center](https://www.cityofsanmateo.org/4294/Online-Permit-Center).

2. **Side yard setback:** Normally 5 ft (R1-C). BUT the rear-one-third exemption (SMMC 27.18.100) allows zero setback if the gazebo is in the rear 1/3 of the lot and eave height ≤ 9 ft, peak ≤ 16 ft. **Must verify both conditions.**

3. **Call 811** before any footing or anchor work — legally required in California.

4. **Check for easements** before siting the gazebo — easements along the side fence line could prohibit the structure entirely. The city's Accessory Buildings page confirms written approval from the utility owner (e.g., PG&E) is required to build within any recorded easement.
